# Smart Study Planner App

## 📌 Description
Smart Study Planner is a web-based application designed to help students organize their study schedule efficiently.

## 🚀 Features
- Add subjects and tasks
- Set study timings
- Track daily progress
- Simple and user-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📅 Future Improvements
- Add login system
- Save data using database
- Notifications/reminders
